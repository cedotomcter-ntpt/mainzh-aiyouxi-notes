# mainzh-aiyouxi-notes

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖技术笔记、工具页面、演示页面等。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开查看，无需额外依赖。

仓库名称中的 `mainzh-aiyouxi-notes` 为归档标识，不代表特定主题或服务。

## 目录结构

```
.
├── README.md               # 本文件
├── index.html              # 仓库入口页面（可选）
├── pages/                  # 归档页面存放目录
│   ├── example-page.html
│   └── ...
└── assets/                 # 公共资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立 HTML 页面，文件名通常反映页面用途。
- `assets/`：存放页面共享的静态资源，如样式表、脚本、图片等。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接通过浏览器打开，或部署到任意静态服务器。
- 页面内容可能涉及技术笔记、工具演示、学习记录等，不针对任何特定域名或网站。
- 归档页面可能随时间更新或移除，恕不另行通知。
